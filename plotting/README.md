# plotting

`/plot.py` is a python file generating the figures in the manuscript.

[`/columnplots.py`](https://github.com/TaoELi/columnplots) is a python plotting script.

`/fig` is the output folder for `/plot.py` containing all the figures in the manuscript.

`/final_results` is the post-processing data for plotting.

Run the following command to obtain all the figures.
```
python plot.py
```
